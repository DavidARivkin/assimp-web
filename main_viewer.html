AssimpView is a stand-alone asset viewer for Windows (doesn't work with WINE, sorry). It loads and displays 3d assets and features stuff like multiple light sources, normal mapping, skinning and high-quality shading. It is intended for both developers and artists to get a quick impression how a model imported with Assimp might look in a realtime environment.

System requirements:

  • Windows 2000, XP, Vista or 7
  • The latest Direct3D 9 Runtime
  • Hardware with support for Shader Model 2.0 (3.0 recommended)
